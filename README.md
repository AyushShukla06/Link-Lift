# LinkLift

This is the LinkLift project repository.

## Features

- Custom link lifting tools built on top of Next.js and various modern tooling.

## Getting Started

1. Instal dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
